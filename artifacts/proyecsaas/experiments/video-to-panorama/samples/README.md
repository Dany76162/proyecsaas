# Videos de prueba

Colocar en esta carpeta videos locales grabados con celular para probar el prototipo.

Formatos sugeridos:

- `.mp4`
- `.mov`
- `.webm`

Recomendaciones de grabacion:

- quedarse quieto en un punto;
- girar lento sobre el propio eje;
- no caminar mientras se graba;
- usar buena luz;
- completar una vuelta;
- evitar movimientos bruscos;
- grabar entre 20 y 60 segundos.

No agregar videos reales al repo.
